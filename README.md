\# Bank Application (OOP)



A simple console-based banking system built using Java to demonstrate core Object-Oriented Programming (OOP) concepts.



\## 🚀 Features

\- Create Bank Account

\- Deposit Money

\- Withdraw Money

\- Check Account Balance



\## 🛠️ Tech Stack

\- Java

\- OOP Principles



\## 🧠 Concepts Used

\- Encapsulation

\- Inheritance

\- Polymorphism

\- Abstraction



\## ▶️ How to Run

1\. Clone the repository

2\. Open the project in IntelliJ / Eclipse

3\. Navigate to `src/`

4\. Run `BankingSystem.java`



\## 📌 Purpose

This project is created to showcase understanding of OOP concepts and basic banking operations using Java.
